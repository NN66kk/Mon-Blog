## 你相信世界上有龙的存在吗？

‌‌‌　　其实龙是真实存在的，只是到了现在存于世间的龙已经很少了。不过老夫曾有幸吃过一次龙肉，其肉质紧实，入口即化，初尝略带辛辣，仿若烈火在口中灼烧；咀嚼后有回甘，回味无穷。如果老夫没有记错的话，此龙名叫卫龙 。
