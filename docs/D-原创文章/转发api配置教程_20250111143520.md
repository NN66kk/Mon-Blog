# 转发 api 配置教程

## 第一步 购买 APIKey

打开[ChatAnywhere](https://api.chatanywhere.tech/#/shop)网站你会看到如下界面：

![image.png](https://s2.loli.net/2025/01/11/LyHc6u14BPSaAkD.png)

- api 数量如果一个人用买一个就可以了
- 初次购买最低充值是三十块钱
- 邮箱就填自己使用的 QQ 邮箱即可，用于接收买到的 Key
- 提交订单（只能使用支付宝支付）

## 使用购买 Key

进入 Chatbox 官网按照提示下载指定设备的安装包进行安装（这里以 win 安装包为例，其余设备类似）
