# 颐和园滑冰初体验

下面的内容有时间会更新。

- [ ] 资费详情

## 资费详情

本来就是奔着滑冰去的，入园门票只买大门票即可，成人 20 元，学生 10 元。

滑冰工具，单人车 50，双人车 100，单人自行车 100。这边建议直接 50 买个双人车，然后到里面可以找人交换。

## 玩法总结

单人车瞎溜达，相当锻炼臂力。

冰上碰碰车：本人的牛突猛进曾把一起来玩的朋友创飞！

接龙游戏：前面一辆自行车，后面挂着一堆单人车，玩的时候随便找，然后挂上跟着跑就行了，非常有意思。

做龙头：这个要求比较多，但也是最好玩的，首先你要有一辆冰上自行车（我们的是找别人交换的），其次你最好是年轻人，这玩意挺消耗体力的，当时我们五个人合作拉车，荣获“最好船长”称号，身后一群追随者~

## 整体体验

八分以上（满分十分），特别是和朋友一起来非常有意思，这算是我遇到的最好玩的团体游戏了！

费用也不贵，你要是有力气，买个车能在里面玩一天，没有时间限制！
