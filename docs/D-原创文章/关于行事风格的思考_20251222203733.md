# 关于行事风格的思考

最近观察各种各样人的行事风格，有了很多思考，这篇文章会持续更新，分析各种风格的好与坏。

- 简单直接型：说什么都是直来直去，这样容易得罪人，但是做起事情来也更加高效，一般都是以解决问题为导向的，需要队友也是这样的，如果一个团队都是这样的人，那这个团队将非常强大。
- 婉转得体型：不管是夸赞的话还是批评的话语，都会说的非常官方和得体，比如批评的话会用一些官方词汇修饰，或者稍微点一下，让人更容易接受。但这中类型的风格一般都是领导的行为，所有在我们这个社会，领导说的话需要反复揣摩，否则就容易会错意。这种行事风格的缺点就是，低效！会把很多时间浪费在修饰语言和理解内容上面，并且下属会错意了可能会造成本该可以避免的错误。优点就是不怎么得罪人，也不会被人抓到把柄。
- 计算得失型：只看当前利益，这种一般考虑的都不长远，只能看到眼前得失。
- 避重就轻型：只做简单的事情，碰见难的会拖或者转嫁给其他人。

以上是简单的总结，以后经过自己见识的增长还会进行重新分类和拓展，敬请关注。
