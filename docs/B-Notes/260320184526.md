# 杠铃卧推运动细节

参考链接：[练胸教程：杠铃卧推的动作 模式、常见错误、发力技巧_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV16i421h71B/?spm_id_from=333.337.search-card.all.click&vd_source=028202a94fa5d8ef90ee83b8f049c8f8)

00:04 原理

- 主要练胸肌（中下束）、三角肌前束、肱三头肌
- 可以使用很大的重量
- 练胸的第一个重量

00:48 动作精讲

- 躺下时杠铃在眼睛的正上方
- 肩胛骨收紧手臂伸直
- 高度比手腕低一点
- 保护杆要比脖子高一点
- 起始姿势锁定肩胛骨
- 双脚往里收踩紧地面，大腿用力绷紧，臀部夹紧，腰腹核心用力收紧，轻微起桥（一个手掌）
- 握距标准为下方杠铃大臂和地面平行的时候小臂和地面垂直
- 握法：杠铃杆放在掌心肉垫位置后稍微内旋，大拇指环扣
- 起杆小臂在杠铃正下方
- 吸气下放，呼气推上去，横杆正对胸点，双手手肘主动往中间夹
- 整个动作小臂始终垂直地面，杠铃的路径为曲线
- 大臂和身体保持七十度夹角即可，全程保持肩胛骨收紧
- 肩胛骨下压，不要耸肩
- 最高点手臂伸直，最低点触胸

06:02 高级技巧

- 手肘往内旋转，试图把杠铃掰弯
- 带个护腕
- 想着把身体往下推，不要想着把杠铃往上推
- 如果被压住了，可以尝试用倾斜杠铃卸下来（前提是不装锁扣）
