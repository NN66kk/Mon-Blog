# 我们上学的目的是什么？

最近一直在思考一个问题，我们到底为了什么而上学，为了找到一个更好的工作？为了认识更多优秀的人？还是为了学到更多的知识？初步想法认为上学不应该是功利的，先插个眼，后续慢慢补充该文的内容...

- 提升自己对这个世界的认知
- 认识更多优秀的人
- 有更多的时间去思考和重塑自己（我认为这个时间非常重要，没有什么比时间更重要的了，就像在外打工的本质就是老板拿钱来换你的时间，请大家一定要尽量留给自己更多的思考的时间，这对提升自己的认知很有帮助）
- 培养自己快速学习一种东西的能力
- 大学课程很多都是点到为止，就是把你领进门，后面更深入的探索取决于你是否感兴趣，想不想付出更多的精力去研究
