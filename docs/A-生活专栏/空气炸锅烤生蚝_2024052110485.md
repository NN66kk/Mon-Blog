# 家用空气炸锅烤生蚝

## 1 前期准备

- 用刷子将生蚝外壳刷洗干净
- 撬生蚝
	- 找到洞捅进去，然后找到个点左右摇摆
	- 生蚝肉取出冲洗干净，裙边需要去掉（有咸腥味）
- 蒜蓉酱调制
	- 大蒜剁碎成蒜蓉
	- 小米辣切碎（可不加）
	- 调味料：盐、白糖、生抽、蚝油、胡椒粉
	- 花生油烧到锅边冒白烟浇入蒜蓉当中

## 2 烧烤步骤

1. 生蚝壳里面放生蚝肉，上面铺上一层蒜蓉酱
2. 3-5 分钟预热烤箱
3. 处理好的生蚝放入烤箱，180 度烤 10 分钟
4. 出锅后撒些葱丝和小米辣即可食用
