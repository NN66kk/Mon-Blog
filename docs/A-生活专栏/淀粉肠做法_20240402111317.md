# 淀粉肠

## 食材

- 鸡胸肉大块
- 猪肥肉增加口感
- 红薯淀粉

## 调料

- 盐
- 白糖
- 花椒粉
- 五香粉
- 大蒜粉
- 耗油
- 鸡精
- 红曲粉
- 姜末

## 制作步骤

1. 以上材料拌到一起打碎成泥
2. 放进冰箱冷冻
3. 红薯淀粉和冰水混合均匀（接近 1:1）
4. 与冷冻好的肉泥混合（稠度放在铲子上下不来就行）
5. 装进肠衣冷水下锅，水开之后等待四十分钟
6. 捞出来过一遍冰水大功告成！
