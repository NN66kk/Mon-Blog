# CLI 与 REPL 的区别

CLI 很简单。  
你输入一条命令。它执行。然后结束。没有记忆。

REPL 不一样。  
你进入一个环境。可以一直说话。它会记住前面的内容。回答会跟着变化。

CLI 更像工具。  
用一次，干一件事。

REPL 更像人。  
你和它一起做事，一步一步推进。

在开发里，区别很明显。  
CLI 用来执行。  
REPL 用来思考。

如果任务很短，用 CLI。  
如果事情复杂，用 REPL。

记住一句话就够了。  
CLI 是“做完就走”。  
REPL 是“一直在场”。
