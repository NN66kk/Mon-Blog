# 微软商店无法访问时的替代下载方案

我的电脑使用的是 **Windows 11**，并且手动关闭了系统自动更新。由于 Microsoft Store 与 Windows Update 机制存在一定耦合，关闭更新后商店往往会出现无法下载或无法访问的情况。如果只是临时需要获取某个应用，其实没有必要恢复系统更新，可以直接绕过商店，通过解析微软商店页面获取 **官方离线安装包**。

具体操作步骤如下：

**第一步：打开微软商店离线包解析站**

访问网站：  
`https://store.rg-adguard.net`

该网站可以解析 Microsoft Store 页面中的安装包下载地址。

**第二步：获取应用的商店页面链接**

在浏览器中进入 Microsoft Store，搜索需要的软件，并打开对应的应用详情页。  
复制该页面的 **完整 URL 地址**。

**第三步：解析下载地址**

将刚才复制的链接粘贴到解析站的搜索框中，并选择 `Retail` 渠道进行解析。

**第四步：下载离线安装包**

页面底部会列出解析得到的安装文件列表。  
找到对应的安装包后 **不要直接点击**，建议右键选择 **“另存为”** 下载到本地。

## 注意事项

-   **下载速度较慢**  
    由于下载链接来自微软全球 CDN，速度可能不稳定，需要耐心等待。
    
-   **页面广告干扰**  
    解析页面存在广告，有时会遮挡下载列表。如果看不到下载链接，可以先关闭广告，链接通常位于页面底部。
    

通过这种方式，可以在 **不恢复 Windows Update 的情况下**，直接获取微软商店应用的官方安装包。
