# GPT 交流技巧

- 提示结构：
	- 任务
		- 讲清楚，别说的太模糊
	- 格式
		- 限制字数
		- 要求分条作答
		- 使用表格输出
	- 角色
		- 措辞用语和详细程度
		- 角色属性、专业不想关或者答案明确的话对输出结果影响不大
	- 例子
		- 不是绝对必要的，但能提高输出质量
- 小技巧
	- 循环提问（GPT 能够理解上下文）
	- 拆分任务
	- 改变人设（三种人设）
		- 友善礼貌
		- 主动发问
		- 给出明确答案
