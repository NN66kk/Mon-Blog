# 等高线地形图

我们可以把等高线地形图想象成一个“切土豆”的过程。

想象一座山，我们把它水平地切成一片一片的（像切土豆片一样），每一片边缘的线条印在纸上，就是等高线。

同一条线上，海拔高度完全一样。

## 怎么看坡度陡不陡？

判断坡度陡不陡，关键看等高线的疏密程度。口诀就是“**密陡疏缓**”。

当等高线非常密集，一条条紧紧挨在一起时，就代表在很短的水平距离内，海拔高度发生了急剧变化，这便是陡坡，爬起来会非常困难。相反，如果等高线看起来很稀疏，线与线之间的距离很大，则说明需要走很长的路，高度才会有少量提升，这就是平缓的坡，行进起来相对容易。

## 怎么分辨山体的不同部位？

**山峰**是山体的最高点，在等高线图上非常容易辨认，它表现为一组闭合的圆圈，如同地图上的靶心。其核心特征是中间的海拔数值最高，并依次向四周降低，代表着地势的顶点。

**盆地或洼地**则与山峰相反，它也是由闭合的等高线圆圈构成，但其地势规律是中间低、四周高，就像一个碗或盆嵌在地面中。为了与山峰区分，图上有时会在最低的闭合圈内标注向下的示坡线，或者通过数值明确显示出中心区域的低洼。

**山脊和山谷**是山体中一对重要且容易混淆的部位，关键在于观察等高线弯曲的凸出方向。**山脊**是山体的高耸部分，像屋脊或动物的脊梁骨，其等高线弯曲部分向海拔数值**低**的地方凸出，是分水岭，雨水会由此向两侧分流，口诀为“凸低为脊”。而**山谷**是两山之间的低凹地带，等高线弯曲部分则向海拔数值**高**的地方凸出，这里是汇水区，河流通常会沿着山谷发育，因此有“凸高为谷”的说法。

**鞍部**是位于两个相邻山峰之间的低洼地带，地形酷似马鞍。在地图上，它表现为两组代表山峰的闭合等高线之间，一片相对平坦或缓坡的区域。这里通常是翻越山岭、连接两个山谷的必经之路，是山脊上最低、山谷中最高的特殊位置。

**陡崖**指近乎垂直的悬崖峭壁。在等高线图上，它的特征非常鲜明，表现为多条等高线汇集并重合在一起，形成一条粗黑的锯齿状线条。这种重合现象只有在地面出现垂直落差时才会发生，直观地展示了地势的急剧变化。
