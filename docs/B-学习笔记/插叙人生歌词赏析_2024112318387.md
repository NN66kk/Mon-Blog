# 插叙人生歌词赏析

## 前言

先说下我对这首歌的认识，起初这首歌刚刚发布的时候我没怎么看歌词，就是觉得旋律挺好提听得就收藏了，半年后当我再次打开这首歌没想到播放量竟然达到了好几百万，我不加思索地看了看歌词，越看越牛逼，最后直接给跪了。这首歌的听感、故事性、韵脚等都达到了顶级水平，真的是大师级的作品！
