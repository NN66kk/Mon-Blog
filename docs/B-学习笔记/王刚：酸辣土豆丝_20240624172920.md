# 王刚：酸辣土豆丝

## 1 前期准备

土豆切丝、葱切成葱花、干辣椒切段；

盐、糖、陈醋、酱油。

## 2 制作步骤

1. 土豆过凉水淘洗干净；
2. 锅中烧开水，放入土豆丝至半透明状捞出，迅速过凉水；
3. 锅中放稍微多点油，放入干辣椒煸香后放入土豆丝；
4. 锅边淋入陈醋和酱油，快速翻炒十来秒；
5. 加入适量食盐和白糖调味；
6. 加入葱花再翻炒几下即可出锅。

## 3 关键点

- 一定要用大火
- 多放油
- 醋和酱油从锅边烹下去
