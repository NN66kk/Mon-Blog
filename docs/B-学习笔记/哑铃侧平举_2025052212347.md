# 哑铃侧平举

## 怎么站

- 决定中束刺激范围的大小
- 上半身略微前倾可以练到后束
- 前倾角度因人而异，可以在用卡片测量法

## 手臂位置

- 手臂必须前移（30°-45°），减少斜方肌发力

## 动作节奏

- 快起慢下效果不行
- 都要很慢效果才是最好的

## 手腕角度

- 水平
- 倒水（微微不建议）
## 握哪里

- 哑铃外部
