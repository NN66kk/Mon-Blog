# Mac 观看杜比视频发紫解决方案

第一步：下载 [IINA](https://iina.io/) 播放器（开源免费）。

第二步：如下图所示，在观看电影界面点击**右下角的三个点**，划到最下面，找到**色调**，拖动**红色圈圈小圆点**改变值，左上角的红色方块区域会显示值，个人认为调整为 **-47** 效果是最好的。

![QUWCa6Dgqs2ZNbO.png](https://s2.loli.net/2025/03/15/QUWCa6Dgqs2ZNbO.png)

第三步：尽情享受杜比视界吧！
